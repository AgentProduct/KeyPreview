# 开发工具集合

## 项目概述

一个基于 React + TypeScript + Vite 构建的现代化前端开发工具集合，提供多种实用的开发辅助工具，帮助开发者提高日常工作效率。

## 技术栈

- **前端框架**：React 19 + TypeScript
- **构建工具**：Vite 7
- **核心依赖**：crypto-js（加密功能）
- **开发工具**：ESLint、Prettier、TypeScript ESLint

## 主要功能模块

### 1. JWT 工具

提供完整的 JWT（JSON Web Token）处理功能：

- **JWT 解码**：解析令牌并自动格式化，支持时间戳字段的人性化显示
- **JWT 编码**：根据指定算法和密钥生成标准 JWT 令牌
- **签名验证**：验证 JWT 令牌签名的有效性和完整性
- **算法支持**：HS256/384/512、RS256/384/512、ES256/384/512
- **密钥格式**：同时支持 PEM 和 JWK 格式的密钥
- **密钥生成**：内置 RSA 密钥对生成功能

### 2. 加密工具

提供常见的加密和解密功能（位于 `modules/encrypt` 目录）

### 3. 格式化工具

代码和数据格式化工具，支持多种格式（位于 `modules/format` 目录）

### 4. 密钥工具

密钥生成和管理工具（位于 `modules/keys` 目录）

### 5. 正则表达式工具

正则表达式测试和调试工具（位于 `modules/regex` 目录）

## 项目结构

```
src/
├── components/      # 通用UI组件
├── modules/         # 核心功能模块
│   ├── jwt/         # JWT处理模块
│   ├── encrypt/     # 加密模块
│   ├── format/      # 格式化模块
│   ├── keys/        # 密钥管理模块
│   └── regex/       # 正则表达式模块
├── tabs/            # 各工具的标签页组件
├── styles/          # 样式定义
├── App.tsx          # 应用入口组件
└── main.tsx         # 应用挂载文件
```

## 开发特性

- **TypeScript 支持**：全项目类型安全，提升开发效率和代码质量
- **模块化设计**：功能模块解耦，便于维护和扩展
- **现代构建工具**：Vite 提供极速的开发体验和构建性能
- **代码质量保障**：集成 ESLint 和 Prettier 确保代码风格一致性
- **响应式设计**：适配不同屏幕尺寸，提供良好的用户体验

## 使用场景

- **开发调试**：快速验证 API 令牌、测试加密算法、调试正则表达式
- **学习工具**：了解 JWT 结构、加密算法原理和实现
- **工作辅助**：格式化 JSON/XML、生成密钥对等日常开发任务
- **教学演示**：展示前端技术栈和工具链的最佳实践

## 技术亮点

- 使用最新的 React 19 和 TypeScript 5.9，充分利用现代前端技术
- 采用模块化架构，实现功能的独立开发和测试
- 遵循最佳实践，包括组件化设计、类型安全和代码质量规范
- 提供友好的用户界面和即时反馈机制

## 扩展指南

1. 在 `modules/` 目录下创建新的功能模块
2. 在 `tabs/` 目录下创建对应的标签页组件
3. 在 `App.tsx` 中注册新的标签页
4. 实现相关功能和 UI 交互
